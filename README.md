## ✅ 과제 구현 목록

**_1) 사진 슬라이드 기능_**

- 화살표 버튼 클릭시 방향에 따라 오른쪽 / 왼쪽 순서대로 사진이 넘어감
- 마지막 사진에서 오른쪽 버튼 클릭 시 첫번째 사진으로 넘어감
- 첫번째 사진에서 왼쪽 버튼 클릭 시 마지막 사진으로 넘어감

**_2) 사진 하단 슬라이드 바에 색상 추가하기_**

- 사진 장수에 따른 슬라이바 생성
- 사진이 바뀔 때 마다 슬라이드바 컬러링
- 화살표 버튼 뿐만 아니라 슬라이드바로 사진 선택 가능
